可以，三种情况要分清楚 `policy.config` 和 `policy.dir`。

我已经给第 2 种补了一个新配置：[config.py](C:/QClaw/FASTER_hy/src/openpi/training/config.py:938)  
配置名是：`pi05_libero_lora_from_libero`

**1. 从 `pi05_base` 开始 LoRA 微调**

这是你现在已有的配置：

```bash
CUDA_VISIBLE_DEVICES=0 WANDB_MODE=offline XLA_PYTHON_CLIENT_MEM_FRACTION=0.9 \
uv run scripts/train.py pi05_libero_low_mem_finetune \
  --exp-name=lora_from_pi05_base \
  --num-train-steps=30000 \
  --overwrite
```

评估/serve 时用 LoRA config：

```bash
CUDA_VISIBLE_DEVICES=0 XLA_PYTHON_CLIENT_MEM_FRACTION=0.9 \
uv run scripts/serve_policy.py policy:checkpoint \
  --policy.config=pi05_libero_low_mem_finetune \
  --policy.dir=checkpoints/pi05_libero_low_mem_finetune/lora_from_pi05_base/29999
```

**2. 从 `pi05_libero` 开始 LoRA 微调**

用我刚加的新配置：

```bash
CUDA_VISIBLE_DEVICES=0 WANDB_MODE=offline XLA_PYTHON_CLIENT_MEM_FRACTION=0.9 \
uv run scripts/train.py pi05_libero_lora_from_libero \
  --exp-name=lora_from_pi05_libero \
  --num-train-steps=30000 \
  --overwrite
```

评估/serve 时也必须用这个 LoRA config：

```bash
CUDA_VISIBLE_DEVICES=0 XLA_PYTHON_CLIENT_MEM_FRACTION=0.9 \
uv run scripts/serve_policy.py policy:checkpoint \
  --policy.config=pi05_libero_lora_from_libero \
  --policy.dir=checkpoints/pi05_libero_lora_from_libero/lora_from_pi05_libero/29999
```

**3. 直接测试官方 `pi05_libero`**

这个不训练，直接 serve 官方 checkpoint：

```bash
CUDA_VISIBLE_DEVICES=0 XLA_PYTHON_CLIENT_MEM_FRACTION=0.9 \
uv run scripts/serve_policy.py policy:checkpoint \
  --policy.config=pi05_libero \
  --policy.dir=gs://openpi-assets/checkpoints/pi05_libero
```

或者等价地用默认 LIBERO policy：

```bash
CUDA_VISIBLE_DEVICES=0 XLA_PYTHON_CLIENT_MEM_FRACTION=0.9 \
uv run scripts/serve_policy.py --env LIBERO
```

然后另一个终端跑 LIBERO 评估：

```bash
source examples/libero/.venv/bin/activate
export LIBERO_CONFIG_PATH=$PWD/third_party/libero
export PYTHONPATH=$PYTHONPATH:$PWD/third_party/libero
export MUJOCO_GL=egl

python examples/libero/main.py \
  --args.task-suite-name libero_spatial \
  --args.num-trials-per-task 50 \
  --args.save-name pi05_libero_official \
  --args.host 127.0.0.1 \
  --args.port 8000
```

关键记忆点：  
`pi05_libero` 官方 checkpoint 用 `--policy.config=pi05_libero`；LoRA 训练出来的 checkpoint 必须用对应 LoRA config 加载。`--num-train-steps=10` 只适合 smoke test，不适合看成功率。
